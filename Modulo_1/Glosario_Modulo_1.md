# Términos y definiciones del Curso de Regresiónes, Módulo 1


## A

- **Análisis de regresión**: Grupo de técnicas estadísticas que utilizan datos existentes para estimar las relaciones entre una única variable dependiente y una o más variables independientes

## C

- **Causalidad**: Una relación de causa y efecto en la que una variable provoca directamente que la otra cambie de una manera determinada
- **Correlación negativa**: Relación inversa entre dos variables: cuando una variable aumenta, la otra tiende a disminuir y viceversa
- **Correlación positiva**: Relación entre dos variables que tienden a aumentar o disminuir juntas
- **Coeficiente de regresión**: Las betas estimadas en un modelo de regresión

## F

- **Función de enlace**: Función no lineal que conecta o vincula matemáticamente la variable dependiente con las variables 
- **Función de pérdida**: Función que mide la distancia entre los valores observados y los valores estimados del modelo

## P

- **Pendiente**: La cantidad que y aumenta o disminuye por el aumento de una unidad de x


## R

- **Recta**: Conjunto de un número infinito de puntos que se extienden en dos direcciones opuestas
- **Regresión lineal**: Técnica que estima la relación lineal entre una variable dependiente continua y una o más variables independientes
- **Regresión logística**: Técnica que modela una variable dependiente categórica en función de una o varias variables independientes


## S

- **Supuestos del modelo**: Afirmaciones sobre los datos que deben ser ciertas para justificar el uso de una determinada técnica de modelización
