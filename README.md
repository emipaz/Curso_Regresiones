# Curso de Regresión

Este repositorio contiene el material de estudio y prácticas del curso de regresión.

- Notebooks en Python para explorar regresión lineal simple y otros modelos de regresión.
- Datos de ejemplo utilizados en las actividades (por ejemplo, archivos CSV de marketing y ventas).
- Material de apoyo en formato Markdown.

El objetivo de este repositorio es servir como espacio de trabajo.
