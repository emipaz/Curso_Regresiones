## Un sindicato nacional de pastores contrata a una empresa de investigación para comprender mejor la relación entre el tiempo de forrajeo, la temperatura, las precipitaciones y los días de sol, y el rendimiento anual de lana de las ovejas. La empresa envía a un investigador a estudiar el mayor rebaño de ovejas del país, recopilar datos y construir un modelo de regresión. ¿Qué supuesto de regresión múltiple violaría este modelo.

- [ ] El supuesto de homocedasticidad 
- [x] La hipótesis de observaciones independientes
- [ ] El supuesto de normalidad multivariante
- [ ] El supuesto de linealidad entre las variables predictoras y la variable dependiente

Buen trabajo
Correcto Si sólo se estudia un rebaño de ovejas, todos estos animales compartirían el mismo hábitat, las mismas condiciones ambientales y, en muchos casos, la misma genética. Esto violaría el supuesto de observaciones independientes.
