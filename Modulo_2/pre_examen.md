## Rellene el espacio en blanco: La línea de mejor ajuste es la que mejor se ajusta a los datos minimizando alguna _____.

- [x] función de pérdida
- [ ] valores residuales
- [ ] valores previstos
- [ ] función de regresión

Correcto
La línea de mejor ajuste es la que mejor se ajusta a los datos minimizando alguna función de pérdida. Para encontrar la línea de mejor ajuste, es necesario medir el error, que es la diferencia entre los valores observados y los valores predichos generados por el modelo.

## ¿Cuál es la suma de las diferencias al cuadrado entre cada valor observado y el valor previsto asociado?

- [ ] Suma de los valores predichos al cuadrado
- [x] Suma de los residuos al cuadrado 
- [ ] Mínimos cuadrados residuales
- [ ] Mínimos cuadrados ordinarios

Correcto
La suma de los residuos al cuadrado es la suma de las diferencias al cuadrado entre cada valor observado y el valor previsto asociado. Los profesionales de los datos utilizan esta suma para captar un resumen del error total del modelo.

## ¿<Cómo determina un profesional de los datos si se cumple un supuesto de linealidad?

- [ ] Confirman si los datos de la coordenada X-Y se parecen a una nube aleatoria.
- [ ] Confirman si los datos de la coordenada X-Y caen a lo largo de una línea curva ascendente.
- [ ] Confirman si los datos de la coordenada X-Y caen a lo largo de una línea curva descendente.
- [x] Confirman si los datos de la coordenada X-Y caen a lo largo de una línea recta.

Correcto
Un profesional de los datos determina si se cumple un supuesto de linealidad confirmando si los datos de la coordenada X-Y caen a lo largo de una línea recta. Se supera un supuesto de linealidad cuando cada variable de predicción X está relacionada linealmente con la variable de resultado Y.

## ¿Cuál de las siguientes afirmaciones describe con exactitud el supuesto de normalidad?

- [ ] La hipótesis de normalidad sólo puede confirmarse mientras se construye un modelo.
- [ ] La hipótesis de normalidad sólo puede confirmarse antes de construir un modelo.
- [ ] El supuesto de normalidad puede confirmarse en cualquier momento durante la construcción del modelo.
- [x] La hipótesis de normalidad sólo puede confirmarse una vez construido el modelo. 

Correcto
El supuesto de normalidad sólo puede confirmarse una vez construido el modelo. Se centra en los errores del modelo, que pueden estimarse mediante los residuos.

## Un profesional de los datos utiliza un diagrama de dispersión para trazar los residuos y los valores predichos de un modelo de regresión para comprobar la homocedasticidad y descubre que se cumple este supuesto. ¿Qué forma tienen los puntos del diagrama de dispersión? 

- [x] Nube aleatoria
- [ ] Línea recta
- [ ] Línea curva
- [ ] Cono

Correcto
Los residuos aparecen como una nube aleatoria de puntos, lo que confirma que la variación de los residuos es coherente o similar en todo el modelo y satisface el supuesto de homocedasticidad.

## ¿Qué tipo de visualización utiliza una serie de gráficos de dispersión que muestran las relaciones entre pares de variables?

- [x] Matriz de dispersión 
- [ ] Matriz residual
- [ ] Matriz lineal
- [ ] Diagrama de dispersión de residuos

Correcto
Una matriz de dispersión utiliza una serie de gráficos de dispersión que muestran las relaciones entre pares de variables. Esto ayuda a los profesionales de los datos a evaluar si existe una relación lineal entre las variables independientes y dependientes.
